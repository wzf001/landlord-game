## 🚀 快速开始

### 方法一：手机/移动端（推荐）

1. 把整个文件夹传到手机，或者部署到服务器
2. 用手机浏览器（微信、Safari、Chrome等）打开 `index.html`
3. 可以添加到主屏幕，像 APP 一样使用！

### 方法二：电脑端

1. 找到 `index.html` 文件
2. 双击打开，或者用浏览器（Chrome、Firefox、Edge等）打开
3. 点击"开始游戏"按钮
4. 开始享受斗地主！

### 方法三：本地服务器（可选）

如果你想使用本地服务器运行：

```bash
# 使用 Python
python3 -m http.server 8000

# 或者使用 Node.js
npx serve

# 然后在浏览器访问 http://localhost:8000
```

## 🎯 游戏规则

### 基本规则
- 一副牌54张，每人17张，3张底牌
- 地主20张牌，农民17张牌
- 地主先出牌，按逆时针顺序轮流出牌
- 可以出单张、对子、三张、顺子、连对、飞机、炸弹等

### 牌型说明
- **单张**：任意一张牌
- **对子**：两张相同点数的牌
- **三张**：三张相同点数的牌
- **三带一**：三张相同点数 + 一张单牌
- **三带二**：三张相同点数 + 一对
- **顺子**：5张或更多连续的单牌（不包括2和王）
- **连对**：3对或更多连续的对子（不包括2和王）
- **炸弹**：四张相同点数的牌（可以打任何非炸弹牌型）
- **王炸**：大王+小王（最大的牌型）

### 叫地主
- 随机决定谁先叫
- 可以选择"叫地主"或"不叫"
- 如果前一家叫了地主，下一家可以选择"抢地主"
- 抢到地主的人获得底牌并先出牌

## 🎮 操作说明

### 叫地主阶段
- 点击"不叫"按钮：不叫地主
- 点击"叫地主"/"抢地主"按钮：成为地主

### 出牌阶段
- **选牌**：点击手牌选中/取消选中
- **提示**：点击"提示"按钮，AI会帮你选择可以出的牌
- **不出**：点击"不出"按钮，跳过这一轮（当上家出牌后才可用）
- **出牌**：点击"出牌"按钮，打出选中的牌

## 💡 游戏技巧

1. **记牌**：记住已经出过的牌，特别是大牌
2. **配合**：如果你是农民，要和另一个农民配合
3. **炸弹**：炸弹很珍贵，不要轻易用掉
4. **顺子**：尽量把散牌组成顺子
5. **叫地主**：手里有大牌（2、王）和炸弹时再叫地主
